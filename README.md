# Blue Velvet Coffee

[Live Website!](https://bluebottlecoffee.com/)

## Background
---
Blue Velvet Coffee is a clone of an American multinational chain of coffeehouses and roastery reserves, [Blue Bottle Coffee](https://bluebottlecoffee.com/). This website is designed on Rails framwork and postgreSQL, React/Redux library as frontend.
<br>
<br>

## Technologies Implemented
---
* Ruby on Rails
* AWS S3
* React/Redux
* HTML and CSS
* render.com(for deployment)
* Javascript
* Jbuilder
<br>
<br>

## Features

## User Authentication - Login, Sign Up, Demo login
---
Blue Velvet Coffee features fully functional user authentication system with all error handling features for incorrect entries. The login and sign up comes down smoothly as a dropdown button. When a user click it gives you a nice modal for login and sign up. Once the user is login then user can write reviews and checkout cart.

### Product Pages/Reviews
---
This features shows nice show page of products based on category. When user click on particular image, it redirect the user to show page of specific product where user can add product to cart if logged in, leave, edit or delete a reviews. The review section has all the reviews ordered chronologically from least to most recent updates. It also gives an idea of average star rating. The review form also have complete error handling, showing an error underneath the input field.

### Cart
---
If user click cart icon in the navigation bar in the header will open the cart modal. The user can update, remove each item from their cart. The price and total quantity icon update dynamically according to each user's cart.
Once the user is done with shopping they can checkout and cart will dynamically clear out and show a "Thank You" modal.
